# ULMS WinForms App - Testing and Debugging Project

This is a clean Windows Forms version of the Umoja Learning Management System (ULMS) created for the software testing and debugging assessment.

## Features

- Login validation
- Student registration
- Course enrolment
- Marks capture
- Average calculation
- Report generation
- Logout
- Input validation and error messages
